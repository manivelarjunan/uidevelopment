Data binding:
What is data binding?
  • Display data by binding controls in an HTML template to properties of an Angular component.
  • Data binding is a mechanism for coordinating what users see, with application data values. While you could push values to and pull values from HTML, the application is easier to write, read, and maintain if you turn these chores over to a binding framework.
  • You simply declare bindings between binding sources and target HTML elements and let the framework do the work.
There are four types of data bindings:-
Types Syntax
String intepolation {{'{{'}} "expression" {{'}'}}{{'}'}}
Proprty binding [target]="expression"
Event binding (event)="statement"
Two way data binding; [(ngModel)]="expression"

String interpolation (or) one way data binding ( {{'{{'}} {{'}'}}{{'}'}} ):

The easiest way to display a component property is to bind the property name through interpolation. With interpolation, you put the property name in the view template, enclosed in double curly braces:. 

{{'{{'}} userName {{'}'}}{{'}'}}

Example:

import {{'{'}}Component{{'}'}} from '@angular/core';
    @Component({{'{'}}
    selector:'my-app',
    template : `
        <p>{{'{{'}} userName {{'}'}}{{'}'}}</p>
         // John
        <p>{{'{{'}} 2 + 5 {{'}'}}{{'}'}}</p>
         // 7'
    {{'}'}})
    export class firstComponent {{'{'}}
        userName :string = 'John';
    {{'}'}}

Property Binding ([]):

Write a template property binding to set a property of a view element. The binding sets the property to the value of a template expression. The most common property binding sets an element property to a component property value. An example is binding the src property of an image element to a component's recipeImageUrl property: property binding as one-way data binding because it flows a value in one direction, from a component's data property into a target element property.
You cannot use property binding to pull values out of the target element. You can't bind to a property of the target element to read it. You can only set it. 

<img [src]="recipeImageUrl">

Example:

import {{'{'}}Component{{'}'}} from '@angular/core';
    @Component({{'{'}}
    selector:'my-app',
    template : `
        <img [src]="heroImageUrl">
        <button [disabled]="flag"></button>
        <div [ngClass]="someClassName"></div>
    {{'}'}})
    export class firstComponent {{'{'}}
        src :string = './images/table.png';
        flag: boolean = false;
    {{'}'}}

Event Binding (()):

Event binding syntax consists of a target event name within parentheses on the left of an equal sign, and a quoted template statement on the right. The following event binding listens for the button's click events, calling the component's addCustomer() method whenever a click occurs:

<button (click)="addCustomer()"> Add Customer <button>

Example:

import {{'{'}}Component{{'}'}} from '@angular/core';
    @Component({{'{'}}
    selector:'my-app',
    template : `
        <input type="text" name="customer name" value ="mike" #customerName />
        <button (click)="updateCustomer(customerName)"> Update Customer </button>
        <p>Customer Name: {{'{{'}} updatedCustomerName {{'}'}}{{'}'}} </p>
         // (initial name) Customer Name:John
         // (after the event) Customer Name:mike
    {{'}'}})
    export class firstComponent {{'{'}}
        customerName : string = 'John';
        addCustomer(customerName:string){{'{'}}
            updatedCustomerName = customerName;
         {{'}'}}
    {{'}'}}

Two way data binding([()]):

Event binding syntax consists of a target event name within parentheses on the left of an equal sign, and a quoted template statement on the right. The following event binding listens for the button's click events, calling the component's addCustomer() method whenever a click occurs:

<button (click)="addCustomer()"> Add Customer </button>

Example:

import {{'{'}}Component{{'}'}} from '@angular/core';
    @Component({{'{'}}
    selector:'my-app',
    template : `
        <p>Customer Name: {{'{{'}} userName {{'}'}}{{'}'}} </p>
        <input type="text" type = "text" value ="mike" [(ngModel)] = "userName" />
         // (initial name) Customer Name:John
        // (As soon as user type customer name updated)  Customer Name:m | mi | mik | mike
    {{'}'}})
    export class firstComponent {{'{'}}
          userName : string = 'John';
    {{'}'}}
