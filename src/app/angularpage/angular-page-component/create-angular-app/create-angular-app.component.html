create angular app:
1. Install node

Go to install node




2. Install angular CLI

After the successful installation, open node CLI then install angular CLI

 > npm install -g @angular/cli


3. Create angular app 
 > ng new my-first-app


5. navigat to the my-first-app folder then enter below command 
 > cd my-first-app
 > ng serve

6.View the app

Open the browser, enter http://localhost:4200/

if you see App works ! then you have created your first angular App successfully. Change the app.component.ts file and change the title, you should see the changes in the app instantly, Also you can add styles in style.css file.
